Catiee is a simple HTML-based web page created to demonstrate the fundamentals of web design and structure. It is built without CSS or JavaScript, focusing only on pure HTML elements. The page includes a header, navigation sidebar, main content section, and footer, along with semantic tags . The purpose of Catiee is to provide a clean and minimal example of how to organize and present content on a webpage using only HTML.
